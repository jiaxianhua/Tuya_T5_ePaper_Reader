# 使用指南

## 按键说明（7 键）
- `UP / DOWN`：列表移动；阅读时上下滚动
- `LEFT / RIGHT`：列表翻页；阅读时前/后翻页
- `MID`：进入目录 / 打开文件 / 确认
- `SET`：横竖屏切换
- `RST`：返回上一级（长按进入百度网盘）

## 文件列表
- 自动挂载 `/sdcard` 并进入列表
- 分页显示目录内容
- 隐藏目录以 `.` 开头的条目会被忽略

## 文本阅读
- 自动换行与分页
- 支持 UTF-8 / UTF-16 / GBK 识别与显示
- 翻页与滚动：
  - `UP/DOWN` 行滚动
  - `LEFT/RIGHT` 翻页
- 断点记录：保存到 SD 卡 `/.sd_reader/progress.bin`

## 图片浏览
- 支持 `BMP / JPG / JPEG / PNG`
- 自动按屏幕比例缩放适配
- 渲染为 1-bit 黑白输出

## 旋转显示
- `SET` 切换 0° / 90°
- 文本与图片界面均支持

## 百度网盘（可选）
- 长按 `RST` 进入授权/列表/详情界面
- 在列表中 `MID` 查看详情，再次 `MID` 下载
- 下载到 `/sdcard/TuyaT5AI`

详见：`docs/wiki/Baidu-NetDisk.md`
